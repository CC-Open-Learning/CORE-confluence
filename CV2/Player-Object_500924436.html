Player Spawner

The player spawner in the Network Manager is used to spawn in a player object when clients connect to the server.

There doesnâ€™t always have to be a player prefab added. Without a player prefab set up clients will spawn in on the same location and see the same view (as long as a camera is present).

Player prefabs can be used to allow the player to interact with the scene, examples from the images include a camera, controller scripts, and local UI.

With a player prefab you can move around the scene and spawn in pre-defined spawn points. While without a player prefab you are locked in the same view as the main camera.

Spawn Points

You can set multiple spawn points around the scene using empty objects and dragging those transform points into the Network Manager inspector.
