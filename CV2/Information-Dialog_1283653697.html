Information Dialog

Information dialogs are simple modals used to provide info/context to the learner about their current task/action. Information dialogs have three labels, (Name label located at the top left of the dialog, an optional title label, and a description label in the content section), and an optional dimmed background.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=4112-1915&t=0Knc2sFibfZ7UaQF-4

Setting up your scene

To get started with implementing information dialogs in your scene

  1. Drag in the information dialog prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() or SetContent() with an InformationDialogSO object to populate content

Properties & Methods

Property/Method

Description

Root

Root VisualElement of the UI

HandleDisplayUI()

Definition:

public void HandleDisplayUI(InformationDialogSO informationDialogSO)

Comments:

  • This method is intended to be a single access method to populate the information dialog with content, and display it

  • Invokes OnDialogShown since Show() is called after content is populated

SetContent()

Definition:

public void SetContent(InformationDialogSO informationDialogSO)

Comments:

  • Populates the information dialog content, but does not call Show()

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the information dialog

  • Invokes OnDialogShown

Hide()

Definition:

public void Hide()

Comments:

  • Hides the root of the information dialog

  • Invokes OnDialogHidden

Scriptable Object

To set up the dialog box, you will need to create an information dialog scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > Dialogs > InformationDialogSO

  • Alternatively, you can create a SO through code and fill out properties from there using 

    InformationDialogSO dialogContent = ScriptableObject.CreateInstance<InformationDialogSO>();
dialogContent.Name = "Name";
dialogContent.Title = "Title";
dialogContent.Description = "Description";
dialogContent.PrimaryBtnText = "Begin";
dialogContent.IsTitleEnabled = true;
dialogContent.IsDescriptionBolded = false;
dialogContent.IsBackgroundDimmed = true;
