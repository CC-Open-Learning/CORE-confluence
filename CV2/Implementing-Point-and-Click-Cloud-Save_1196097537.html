CORE Framework : Implementing Point and Click Cloud Save

Created by Ryan Wallace on Jun 27, 2024

Cloud Save and the Point and Click Packages are to separate packages that are accessible at VAR Lab, in this documentation I will be going over how I combined the two to allow for a saving and loading system into a sample point and click scene.

We have created a script called Serialized NAV Properties, when applying a GUID System and Save System in the point and click scene.

Add the NAV Properties Script to the main camera object in the scene producing the following:

This Shows it is saving the rotation and position to cloud save underneath drag the POV Camera into the POV Cam part of the script allow the script to serialize the POV cameras vertical and horizontal values into cloud save. In the GUID System click the button to updated tracked objects this configure the script into a tracked component for the Cloud save system.

After that cloud save and load should be set up in a point and click environment, Next implement the button canvas creating 3 buttons save, load, and delete. When the save button is clicked it should save the position, rotation and POV camera view coordinates. When load it will take what was saved and re-position the camera back to the previous place. Delete will delete all saves that have been sent to the cloud save resetting you to zero.

Add the Menu Toggle script onto the menu manager attached to the button canvas. Then make sure the scene has an Event System. With this configuration in the scene whenever you click the ESC key it will show the button canvas and allow you to interact with it in a point and click environment.
