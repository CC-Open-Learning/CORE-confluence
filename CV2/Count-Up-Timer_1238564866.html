Timer

The timer is a simple elapsed time counter that can be expanded/collapsed. Timers have no customization available besides whether it starts open and/or paused.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=3100-6049&t=MRXTZUTJQKK1hhUj-4

Setting up your scene

To get started with implementing timers in your scene

  1. Drag in the count up timer prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() with the open state, and pause state of the timer

HandleDisplayUI()

Definition:

public void HandleDisplayUI(bool startOpened = true, bool startPaused = false)

Comments:

  • This method is intended to be a single access method to reset the timer, show the root, and open/start the timer based on the incoming startOpened/startPaused bools

  • Invokes OnTimerShown

ToggleTimer()

Definition:

public void ToggleTimer()

Comments:

  • Toggles the timer container and arrow to its open/closed state by toggling USS classes

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Pause()

Definition:

public void Pause()

Comments:

  • Pauses the timer by setting IsPaused to true

  • Invokes OnTimerPaused

Resume()

Definition:

public void Resume()

Comments:

  • Resumes the timer by setting IsPaused to false

  • Invokes OnTimerResumed

SetElapsedTime()

Definition:

public void SetElapsedTime(double newTime)

Comments:

  • Sets the elapsed time for the timer

ResetTimer()

Definition:

public void ResetTimer()

Comments:

  • Resets the timer. Sets ElapsedTime to 0, IsPaused to true, IsOpen to false, and applies USS classes for closed state

  • Invokes OnTimerReset

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the timer

  • Invokes OnTimerShown

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Hide()

Definition:

public void Hide()

Comments:

  • Hides the root of the timer

  • Invokes OnTimerHidden

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()
