CORE Framework : Waypoint Customization

Created by Amila Abeygunasekara, last modified on Jan 15, 2024

A waypoint comes with a child GameObject named Marker. The marker is the GameObject that holds the waypoint icon.

The motivation behind making the marker a child is to give the user an option to shift the marker relative to the waypoint without changing the waypoint’s position.

The Marker GameObject comes with the following components.

For customizations, we are mostly interested in Marker and Sprite Renderer components.

Editing Marker Position, Rotation & Scale

To edit the marker’s transform variables, all you have to do is edit the values in the Transform component.

NOTE: When clicking on a Marker, the player will walk into the parent Waypoint’s location, not the Marker’s location.

Editing Marker Icon

You can update the marker icon by serializing a new sprite into Sprite Renderer’s Sprite field.

Updating Colors

Highlight and Default colors can be set by adjusting the respective field in the Marker component.
