CORE Framework : Using the Objectives System

Created by Jordan Robinson on Jul 25, 2022

In this article weâ€™ll cover some examples of how to use the objective system to add or remove objectives as well as change their status and subscribe to the different events the system has.

Adding Objectives

In order to add objectives, we first need to call ObjectiveManager.InitializeObjectives(); to set up the objective dictionary for this session.

This is required explicitly rather than automatically cached to force devs to think about when they need to create a new set of objectives. For example if you move from one level to another you may want to call it again to flush the old objectives out if they are no longer relevant.

If we want to add an objective to our list, we can use ObjectiveManager.CreateObjective(string objectiveName); to create an objective. This will add a new objective to our dictionary and set it up with default values.

Setting Objectives

To set an objective or create an objective with non-default values, we can call SetObjective instead. 

ObjectiveManager.SetObjective("Task", CompletionStatus.Completed, AnswerStatus.Incorrect);

SetObjective will create an objective by the name provided if one does not already exist.

Checking Objectives

To check if an objective is in a specific state, we can call CheckObjective.

var done = ObjectiveManager.CheckObjective(taskName, CompletionStatus.Incomplete, AnswerStatus.Incorrect);

If we need to check if all of our objectives are complete, we can use CheckCompletionsForAllObjectives.

For example, if we wanted to check that all objectives have been completed, we could write:

var allDone = ObjectiveManager.CheckCompletionsForAllObjectives(CompletionStatus.Completed);

Getting Objectives

If we need to actually get the objective instances that satisfy a given set of conditions, we can do so using GetObjectivesWhere.

var predicate = new Predicate<Objective>(e => e.CompletionStatus == CompletionStatus.NotStarted);

var matching = ObjectiveManager.GetObjectivesWhere(new Func<Objective, bool>(predicate));

In this example, GetObjectivesWhere will return an IEnumerable of Objective objects that have a CompletionStatus of NotStarted.
