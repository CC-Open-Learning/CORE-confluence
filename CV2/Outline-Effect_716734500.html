Summary

The Easy Performant Outline Unity asset provides visual effects for highlighting and outlining objects in 2D and 3D Unity projects.

While all of Unity’s modern render pipelines are supported, CORE packages and DLX projects use the Universal Render Pipeline.

https://bitbucket.org/VARLab/easy-performant-outline/src/main/

Installation

UPM Package

Easy Performant Outline is a third-party asset developed by Pirate Parrot. VARLab maintains and internally distributes EPOutline as a UPM-compatible package through a Bitbucket repository.

In the Packages/manifest.json file of a DLX Unity project, add the following line to dependencies:

"com.pirateparrot.outline": "ssh://git@bitbucket.org/VARLab/easy-performant-outline.git#3.4.3"

This will install the specific stable version of Easy Performant Outline as a UPM package into the Unity project.

Shaders and Materials

Navigate to the project folder structure and go to Packages → Easy Performant Outline → Package Resources and select the EP Outline Shaders.unitypackage in order to import the required shaders and materials.

Render Features

Most projects built on the CORE Framework will be using the Universal Render Pipeline (URP) for Unity. The UPM package for this Outline feature is specifically intended for use with the URP.

Review Unity’s documentation on getting started with the Universal Render Pipeline if it is not already installed in the project.

Navigate to Tools → EP Outline → Setup to open the setup dialogue, then select URP as the setup type. No projects are currently using the DOTween animation engine, so leave the DOTween support checkbox blank.

The dialogue will dynamically update itself based on the correctly configured steps in the process:

  1. Install Universal Render Pipeline asset (consult Unity documentation linked above)

  2. Add the URP_OUTLINE compiler flag by clicking the Add button in the dialogue

  3. Create at least one renderer asset (this step will already be completed for existing projects using URP)

  4. Add the URP Outline Feature as a Renderer Feature to each of the renderer assets. This can be done from the setup dialogue, or in the inspector window when selecting a renderer asset in the project.

RenderGraph Compatibility

Unity 6 encourages the use of the RenderGraph API, though the Easy Performant Outline package does not currently support this.

After installing the package into a project using Unity 6, navigate to Edit → Project Settings → Graphics → Render Graph and check the box for Compatibility Mode (Render Graph Disabled). This will allow the outline features to be rendered properly, until EPO is updated to support Render Graph.

You will likely see the following warning message in the console once Compatibility Mode is enabled:

The project currently uses the compatibility mode where the Render Graph API is disabled.
Support for this mode will be removed in future Unity versions. 
Migrate existing ScriptableRenderPasses to the new RenderGraph API. 
After the migration, disable the compatibility mode in Edit > Projects Settings > Graphics > Render Graph.
