CORE Framework : Vive Cosmos Plugin

Created by Lisa Nguyen, last modified by Anzhelika Kostyuk on Mar 06, 2023

Source: Installing VIVE OpenXR PC VR Plugin - Developer Resources

To use the VIVE Cosmos you need to add their plugin. Go under Edit → Project Settings → Package manager then enter in this information in scoped registries:

Name: VIVE

URL: https://npm-registry.vive.com

Scopes: com.htc.upm

After that go into your Package Manager and under ‘My Registries’ you should see all the available packages by HTC Corporation. Install the 'VIVE Wave OpenXR Plugin - Windows' plugin.

Go back to Edit → Project Settings → XR Plugin Management>OpenXR and add in the Vive Cosmos controller like below:
