CORE Framework : Bayou Transport

Created by Lisa Nguyen, last modified on Jul 22, 2022

Bayou is the transport used for WebGL builds.

GitHub: FirstGearGames/Bayou: A WebGL transport for Fish-Networking. (github.com)

Download the Bayou code from GitHub. Unzip the file and open the folders until you see Bayou folder and Bayou.meta file- copy these two files.

In another explorer window open your project folder, go into Assets > FishNet > Plugins and paste your copied files into the folder.

You should now be able to find the Bayou script when you search for it under ‘Add Component’.
