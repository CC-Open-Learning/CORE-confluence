CORE Framework : VehicleHandle

Created by Geyang Liu on Aug 18, 2022

The vehicle handle is used to assist boarding and controlling vehicles in VR mode. To set it up, create a grabbable gameobject and assign the VehicleHandle script to it. In the public fields, select a handle type and assign the vehicle it will be binded to.

When a hand grabs onto the handle, the handle will accept input from the handâ€™s controller and pass the input to the vehicle body.
