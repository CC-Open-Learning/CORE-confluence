Helper & Extension Methods

VELCRO UI has various extension and helper scripts which are public static methods. Most helper scripts are used for specific UI prefabs, but VisualElementExtensions.cs, TextElementExtensions.cs, PositionHelper.cs, and StyleHelper.cs have methods for common use cases in many UI scripts.

SetAbsoluteVerticalPosition()

Definition

public static void SetAbsoluteVerticalPosition(VisualElement visualElement, Align alignment)

  • Sets the VisualElement.style.top and VisualElement.style.bottom properties of the incoming VisualElement to position the element to the Align enum

  • All UI elements inherit from VisualElement

Parameter(s)

VisualElement The element to change the top/bottom properties of

Align The alignment to set

//Supported alignment
public enum Align
{
    FlexStart, //Horizontally centered, vertically pinned to the top of the screen
    Center,    //Horizontally centered, vertically centered
    FlexEnd    //Horizontally centered, vertically pinned to the bottom of the screen
}

Example

/// <summary>
/// This method is intended to be a single access public method to populate the notification with text
/// and display it at the desired position (top, center, or bottom). This method accepts a SO rather 
/// than individual properties if DLX have the ability to predefine their UI contents
/// </summary>
/// <param name="notificationSO"></param>
public void HandleDisplayUI(NotificationSO notificationSO)
{
    PositionHelper.SetAbsoluteVerticalPosition(notification, notificationSO.Alignment);
}

SetUITopAnchor()

Definition

public static void SetUITopAnchor(VisualElement visualElement, TopAnchor anchor)

  • Sets the properties of the incoming VisualElement to position the element to the TopAnchor enum

    • VisualElement.style.top

    • VisualElement.style.bottom

    • VisualElement.style.alignSelf

  • All UI elements inherit from VisualElement

Parameter(s)

VisualElement The element to change the top/bottom/alignSelf properties of

TopAnchor The alignment to set

//Supported alignment
public enum Align
{
    TopRight = 0, //Anchored to the top right corner
    TopLeft = 1   //Anchored to the top left corner
}

Example

/// <summary>
/// Populates the task list from the content defined in the scriptable object
/// </summary>
/// <param name="taskListScriptableObject">Task list scriptable object</param>
public void PositionTaskList(TaskListSO taskListScriptableObject)
{
    PositionHelper.SetUITopAnchor(taskList, taskListScriptableObject.Alignment);
}
