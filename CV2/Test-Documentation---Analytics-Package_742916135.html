This is a system test to ensure the system is successfully communicating with the 3rd party API.

  1. Start the analytics package in play mode

  2. Log in to PlayFab, and go to the title's PlayStream Monitor.

  3. Click the login button

  4. Click the Progression button

  5. Click the Section button
