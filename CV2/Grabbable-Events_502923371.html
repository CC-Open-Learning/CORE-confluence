CORE Framework : Grabbable Events

Created by Geyang Liu, last modified on Aug 04, 2022

The native Autohands asset only supports controller interactions with the grab and trigger functions. CoreV2 uses the class GrabbableEventManager and the interface IGrabbableEvents to add additional controller interactions. Follow the following steps to set it up:

  1. Let the object script that needs to interact with the controller implement the IGrabbableEvents interface. Implement the interactions in the interface methods.

2. For each controller that needs to interact with objects, attach a GrabbableEventManager script to the gameobject. Assign the correponding Autohand Hand and Controller link in the Hand and Controller Link fields. For each controller button, add an entry in the Button Events list, select the button and assign thefunction it triggers.
