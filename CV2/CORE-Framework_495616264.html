CORE Framework : CORE Framework

Created by Jordan Robinson, last modified by David Inglis on Oct 11, 2024

The CORE Framework (CORE) is a pedagogy-first architecture and toolset for developing Digital Learning Experiences (DLX), primarily in the Unity 3D game engine.

Purpose

This Confluence space holds documentation and research for CORE Framework.

Delivery

CORE is delivered as a series of Unity packages, and accompanying web services and API. CORE becomes not a monolithic Unity project with all DLX, but instead a collection of single-purpose packages which can imported as needed into DLX project workspaces.

The https://varlab-dev.atlassian.net/wiki/spaces/CV2/pages/569147393/List+of+CORE+Features?atl_f=content-tree articulates the various packages and services that make up the CORE Framework.
