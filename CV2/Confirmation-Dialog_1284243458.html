Confirmation Dialog

Confirmation dialogs are simple modals that prompt the learner to proceed with an action, cancel, or close. The prompt/action can be destructive as the primary and secondary buttons can be changed.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=4112-1915&t=0Knc2sFibfZ7UaQF-4

Setting up your scene

To get started with implementing confirmation dialogs in your scene

  1. Drag in the confirmation dialog prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() or SetContent() with a ConfirmationDialogSO object to populate content

Properties & Methods

Property/Method

Description

Root

Root VisualElement of the UI

HandleDisplayUI()

Definition:

public void HandleDisplayUI(ConfirmationDialogSO confirmationDialogSO)

Comments:

  • This method is intended to be a single access method to populate the confirmation dialog with text, buttons, and display it

  • Invokes OnDialogShown since Show() is called after content is populated

SetContent()

Definition:

public void SetContent(ConfirmationDialogSO confirmationDialogSO)

Comments:

  • Populates the confirmation dialog text, and buttons, but does not call Show()

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

SetCloseButton()

Definition:

public void SetCloseButton(bool isCloseBtnVisible)

Comments:

  • Sets the closeBtn to DisplayStyle.Flex or DisplayStyle.None depending on incoming isCloseBtnVisible

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the confirmation dialog

  • Invokes OnDialogShown

Hide()

Definition:

public void Hide()

Comments:

  • Hides the root of the confirmation dialog

  • Invokes OnDialogHidden

Scriptable Object

To set up the dialog box, you will need to create a confirmation dialog scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > Dialogs > ConfirmationDialogSO

  • Alternatively, you can create a SO through code and fill out properties from there using 

    ConfirmationDialogSO dialogContent = ScriptableObject.CreateInstance<ConfirmationDialogSO>();
dialogContent.Name = "Name";
dialogContent.Description = "Description";
dialogContent.PrimaryBtnType = ButtonType.Negative1;
dialogContent.PrimaryBtnText = "Negative";
dialogContent.SecondaryBtnType = ButtonType.Secondary1;
dialogContent.SecondaryBtnText = "Cancel";
dialogContent.IsCloseBtnVisible = true;
dialogContent.IsBackgroundDimmed = true;
dialogContent.IsDescriptionBolded = true;
