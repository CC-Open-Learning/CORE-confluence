CORE Framework : PC Player

Created by Geyang Liu, last modified by Anzhelika Kostyuk on Mar 06, 2023

The PCPlayer prefab is used for navigating and interacting with scenes in PC mode. This prefab is mostly identical to the VRPlayer prefab, preserving most of the functionalities from Autohands. There are only the following listed changes to switch to PC controls.

Remove VR device binding

The XR Hand Offset component under the prefab root and the XR Hand Player controller link component under AutoHandPlayer is removed. These two components were used to bind the player's hands to the VR devices.

Add PC input module

A Player Input component and a Player Controller script are added to the AutoHandPlayer gameobject. These two components allow user to control the player through mouse and keyboard.
