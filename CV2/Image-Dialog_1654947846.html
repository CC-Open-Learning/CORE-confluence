Image Dialog

Image dialogs are modals used to provide info/context to the learner about their current task/action, along with up to 2 images. Image dialogs have 5 labels, (Name label located at the top left of the dialog, an optional title label, a description label in the content section, along with a sub description label and a note label below), and an optional dimmed background.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=6187-22505&t=BFjdaOBmxuX7x7B3-0

Setting up your scene

To get started with implementing information dialogs in your scene

  1. Drag in the image dialog prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() or SetContent() with an ImageDialogSO object to populate content

Description

Root

Root VisualElement of the UI

HandleDisplayUI()

Definition:

public void HandleDisplayUI(ImageDialogSO imageDialogSO )

 

Comments:

  • This method is intended to be a single access method to populate the information dialog with content, and display it

  • Invokes OnDialogShown since Show() is called after content is populated

SetContent()

Definition:

public void SetContent(ImageDialogSO imageDialogSO )

 

Comments:

  • Populates the image dialog content, but does not call Show()

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Show()

Definition:

public void Show()

 

Comments:

  • Shows the root of the image dialog

  • Invokes OnDialogShown

Hide()

Definition:

public void Hide()

 

Comments:

  • Hides the root of the image dialog

  • Invokes OnDialogHidden

Scriptable Object

To set up the dialog box, you will need to create an information dialog scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > Dialogs > ImageDialogSO

  • Alternatively, you can create a SO through code and fill out properties from there using

            imageDialogSO = ScriptableObject.CreateInstance<ImageDialogSO>();
        imageDialogSO.Name = "Name";
        imageDialogSO.Title = "Title";
        imageDialogSO.Description = "Description";
        imageDialogSO.PrimaryBtnText = "PrimaryBtn";
        imageDialogSO.IsBackgroundDimmed = true;
        imageDialogSO.SubDescription = "SubDescription";
        imageDialogSO.Note = "Note";
        imageDialogSO.DialogImage = dialogImage;
        imageDialogSO.NoteImage = noteImage;
