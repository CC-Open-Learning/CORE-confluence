Gradient Element

The gradient element is similar to a visual element but allows for a gradient background. This is used in VELCRO for places such as the count up timer.

Using the Gradient Element

To get started using the gradient element in the UI builder

  1. Drag and drop the control from the Library panel in “Library > Project > Custom Controls > VARLab > Velcro > Gradient Element” to the hierarchy

    1. By default, the element does not show anything in the viewport but once you add a width and height to it (or set grow to 1) you will see something like this:

  1. To change the gradient element's colours, create or add an existing USS file in the StyleSheets panel

  2. Create a USS class, and apply it to the gradient element

    1. You can set the variables to either a HEX value or an RGB value.

.gradient-element {
	--top-left-colour: #020024;
	--bottom-left-colour: #020024;
	--top-right-colour: rgba(0, 212, 255, 1);
	--bottom-right-colour: rgba(0, 212, 255, 1);
}

Adding Border Radius and Colours

Since the custom control uses Unity’s Mesh API internally to render the gradient colours, UI toolkit features like border colour and radius do not work directly on gradient elements.

To add a border radius or colour to a gradient element,

  1. Create a new parent VisualElement for the gradient element

If you have a specific width and height applied to the gradient element, it would be best to apply those dimensions to the parent visual element and set the gradient element’s grow value to 1.

  1. Once you have the gradient element inside of another VisualElement, set the Display properties of the parent visual element and set the overflow property to hidden.

  1. Set the border radius and colour of the parent VisualElement to whatever you would like
