Unity Analytics (How does it work?)

The Unity Engine provides an analytical engine that can be referenced in a script or added to a component. Custom events can be implemented on Unity, and the data is displayed on a Unity Dashboard. These custom events are the analytics to help influence decisions from the captured data using the dashboard funnels.
