ProgressContainer

Progression events are used to track user progress through a given part of the simulation. These are meant to be used inside of an evaluation/quiz to track how the user navigates and performs inside of that Segment.

public ProgressContainer(bool? correct, string progressName, int SegmentNo)

Parameters:

correct - Nullable Bool. True is correct, false is incorrect, null represents a neutral state where the user has progressed outside of a scenario where there are correct and incorrect options.

progressName - a string representation of the location in the simulation where the user is when the progression event is fired. Used for sorting and querying data.

SegmentNo - an integer that represents the Segment that the event is taking place in

Event Body:

Status: (string) Correct/Incorrect/null

Event: (string) relative information about the current progress

Segment#: (int) indicates the Segment number where the event is taking place
