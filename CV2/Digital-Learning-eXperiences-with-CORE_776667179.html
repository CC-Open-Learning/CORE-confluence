CORE Framework : Digital Learning eXperiences with CORE

Created by David Inglis, last modified by Talon Ernst on Sept 06, 2024

Summary

Digital Learning Experiences (DLX) in Unity can take advantage of the numerous utilities provided by the Unity platform and third-party developers. These utilities may come in the form of packages that can be imported into a Unity project.

The https://varlab-dev.atlassian.net/wiki/spaces/CV2 is a collection of packages developed by VARLab (CVRI?) specifically intended to support DLX development in Unity. Many of these packages are Unity-specific, while some, such as the Cloud Save web API, are platform-agnostic support services that can be accessed from within Unity and other development contexts.

Unity Project Setup

Development teams can create a Unity project using the DLX Project Template as a project base. This template contains boilerplate code for configuring and integrating key services from the CORE Framework library.

https://bitbucket.org/VARLab/dlx-project-template/src/main/

Navigation

Navigation in the CORE Framework consists of different packages to allow for fluid movement of camera and the player character allowing developers to implement navigation within a scene more efficiently. Consisting of Freeform Navigation and Point and Click Navigation.

Point and Click Navigation

The Point and Click Navigation is a package that can be accessed in the DLX Project Template. This package contains a prefab for the Point and Click Camera and reference scenes and plains that allow developers to re create Point and Click Navigation within any scene. This removes the needs from the developers to re create a point and click navigation movement system from the beginning but instead relying on it being already an easy to implement package.
