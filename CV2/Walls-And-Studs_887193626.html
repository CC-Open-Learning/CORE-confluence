CORE Framework : Walls And Studs

Created by Ali Kaya (Unlicensed) on Dec 14, 2023

Overview

The Procedurally Generated Worlds project aims to facilitate the procedural generation of buildings by incorporating essential features such as Walls, Windows, Doors, Studs, Pillars, and Floors. These elements collectively enable the creation of diverse building structures within the procedural generation system.

Key Features

Pillars

Pillars serve as crucial points for defining walls within the building structure. They act as connected points that facilitate the creation of walls between them. To manually connect two pillars:

  • Obtain two pillar prefabs.

  • In the Inspector, select one of the pillars.

  • Navigate to the "LookAt.cs" Script component.

  • Under the "target" section, choose the other pillar prefab.

  • This linkage enables the creation of walls between the selected pillars during play mode.

Studs

Studs, originally designed for the Trades Electrical project, are utilized to convert a wall object. By accessing the "WallToStuds.cs" script component in the Inspector for a wall object, the "Place Studs" checkbox can be toggled to transform the wall into studs.

Walls

Walls are dynamically generated when two pillars are connected. They can be modified to incorporate windows or doors by interacting with the "WindowAndDoorManager.cs" script. Key functionalities include:

  • Transformation into studs

  • Addition of windows or doors by connecting corresponding objects to the wall

  • Checkbox options such as "Is Door," "Done," and "Check Collision" for managing window and door placement and collision detection

Door and Windows

Door and Window objects are represented as default Unity 3D rectangle objects. The "WindowAndDoorManager.cs" script recognizes and manages these objects, allowing the designation of windows or doors through the "Is Door" and "Done" checkboxes.

Floor

Floors are generated via the Grid Building System and are plane objects placed within the grid. They serve as anchor points for walls, studs, and interior design elements within the building structure.

Usage Instructions

  1. Pillar Connection

    • Obtain pillar prefabs.

    • Connect pillars using the "LookAt.cs" script.

  2. Stud Placement

    • Access the "WallToStuds.cs" script for a wall object in the Inspector.

    • Toggle the "Place Studs" checkbox to transform the wall into studs.

  3. Wall Customization

    • Utilize the "WindowAndDoorManager.cs" script for wall modifications.

    • Manage window and door addition using checkboxes and object snapping.
