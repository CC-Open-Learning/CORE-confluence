Created by Ryan S (Unlicensed) on Aug 03, 2023

This is an example template of a Code Coverage Report. This page should NOT be used to report your project’s code coverage. Instead, use your project’s Code Coverage report page, found on the project’s Confluence space.

To add a report to your project’s Confluence space, create a new page and select the Code Coverage Report Template. Be sure to add the page as a child of your project.
