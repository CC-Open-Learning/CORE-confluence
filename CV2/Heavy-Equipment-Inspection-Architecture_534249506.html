CORE Framework : Heavy Equipment Inspection Architecture

Created by Kristian Biviens, last modified by David Inglis on May 03, 2023

Heavy Equipment Inspection System Package

Inspection Manager

The inspection manager will hold the groups of the inspections. The InspectionManager class will hold an array of Inspection Elements. It will provide the opportunity to automatically find and aggregate Elements or allow the developer to add them manually. The learning experience inspections will be divided into multiple groups to better organize the data. We want to refrain from having all of the inspections in one group.

InspectionElement

The InspectionElement class will contain an array of ScriptableObjects called conditions.

Element Condition

The object Element Condition will contain the conditions of all inspected objects. This will allow us to keep information related within this one area which will be used by ScriptableObjects and by the InspectionElement class.

Unity Engine

This describes how the system uses types provided by the Unity Engine.

Scriptable Object

The ScriptableObject is a data container that can be used to save different amounts of data while being independent of class instances. Data and logic can be predefined during editor time. They are very similar to class instances as assets.
