Toolbar Collapse

Toolbar Collapse is a controllable version of the toolbar that allows users to show/hide its contents. It allows users to select between a set number of options conveniently contained on the side of the screen. Each option can be represented by one of two types of buttons:

  • Complex Button - Icon + Text

  • Simple Button - Icon only

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=3486-17523&m=dev

Setting up your scene

To get started with implementing Toolbar Collapse in your scene

  1. Drag in the Toolbar Collapse prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() or SetContent() with a ToolbarSO object to populate content

Scriptable Object

To set up the Toolbar Collapse, you will need to create a ToolbarSO scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > ToolbarSO

  • Alternatively, you can create a SO through code and fill out properties from there using

    ToolbarSO toolbarContent = ScriptableObject.CreateInstance<ToolbarSO>();
toolbarContent.Buttons = new List<ToolbarButton>();

toolbarContent.Buttons.Add(new ToolbarButton(icon, text)); // Adds a complex button
toolbarContent.Buttons.Add(new ToolbarButton(icon, null)); // Adds a simple button
