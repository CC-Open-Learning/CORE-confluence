Table Category

Table Category is a class representing a category on the table. It contains properties that allow users to modify individual categories.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=4110-1521&m=dev

TableCategory()

Definitions:

public TableCategory()

Comments:

  • Constructs a blank table category

  • Used internally by the table class

  • Constructing a table category will not give any special functionality. The proper way of adding table categories to a table is via the AddCategory() method

AddEntry()

Definition:

public void AddEntry()

Comments:

  • Adds a new entry to the category

  • Invokes OnEntryAdded

RemoveAllEntries()

Definition:

public void RemoveAllEntries()

Comments:

  • Removes all entries from the category

  • Invokes OnEntryRemoved for as many columns as there were

RemoveFromTable()

Definition:

public void RemoveFromTable()
