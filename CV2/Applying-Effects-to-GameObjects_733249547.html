CORE Framework : Applying Effects to GameObjects

Created by David Inglis on Jun 13, 2023

The Easy Performant Outline package can be used to apply outline and highlight effects to 2D and 3D objects in a Unity scene.

The author of the EP Outline package, developer Pirate Parrot, has published install and usage documentation as a Google Doc

Setup

Camera Settings

Locate the Main Camera in the scene and add an Outliner component.

The Mask field of the Outliner indicates which layers these global settings apply to; layers can be specified for each object on their corresponding Outlinable component described below, or the mask can be set to all in order to apply to all Outlinables.

Object Setup

An Outlinable component can be added to any game object to apply outline and highlight effects to one or more Renderers.

For a simple object, such as a 3D primitive cube, an Outlinable component would be added to the cube GameObject and would apply the effects to the mesh renderer for the cube.

The collection of Renderers shown at the bottom of the component indicates which renderers are going to be affected by these specific effects settings. By default, child renderers will be included in this collection when an Outlinable component is added to the parent.

Complex objects can be configured so that the same Outlinable properties are applied to all meshes

Considerations

As warned in the Outlinable component, setting custom dilate and blur settings on each Outlinable is computationally expensive. Instead, modify the settings of the Outliner on the main camera, or create additional Outliners, in order to customize the appearance of the outline for one or more layers, then use the Outline Layer property on the Outlinable to organize objects into groups by layer.
