Task List

Task lists are used to track learner progress on their current objective. Task lists were used in AMB seek and find mode.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=3681-7535&t=7BW93pYlUSJahL5r-4

Setting up your scene

To get started with implementing task lists in your scene

  1. Drag in the task list prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Add a reference to the notification prefab from your project scene hierarchy. If the notification prefab has not been added to you project, add it by following these instructions: Notifications

  4. Hook up UnityEvents as required

    1. You will need to use a TaskListSO to create a task list using CreateTaskListFromSO()

Scriptable Object

To set up the task list you will need to create a task list scriptable object asset.

  • Right click inside the folder you want to save the scriptable object and select Create > ScriptableObjects > TaskListSO

  • You can also create a SO through code and fill out the properties as shown below

TaskListSO taskList = ScriptableObject.CreateInstance<TaskListSO>()
taskList.ListTitle = "List Title";
taskList.ProgressText = "Completed";
taskList.ListDescription = "List Description";
taskList.Alignment = TopAnchor.TopLeft;

// Create the tasks for the SO
var task1 = new TaskListSO.Task();
task1.TaskName = "Task 1";
task1.TaskHint = "Task 1 hint";
var task2 = new TaskListSO.Task();
task2.TaskName = "Task 2";
task2.TaskHint = "Task 2 hint";

// Add the tasks to the SO
taskList.Tasks = new()
{
    task1,
    task2,
    task3,
    task4
 };

  • Task can also be added using the AddTask() after the list has been created.
