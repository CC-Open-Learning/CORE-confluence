Introduction

This page covers how to set up Unity’s Code Coverage package. To begin, make sure Unity is installed, and open a project containing tests you wish to use. If you don’t have tests for your current project, refer to the testing documentation located HERE. If you don’t have a project and wish to learn code coverage, download the example project HERE.

Video Step-by-Step Guide

This video is a visual representation of the instructions below. You may skip it and follow the text instructions instead.

CodeCoverageSetupMk2.mkv

Installation

  • Open the Unity editor.

  • Use the top bar and navigate to Window > Packet Manager.

  • Change the Packages from Packages: In Project to Packages: Unity Registry.

  • Search “Code Coverage” in the search bar at the top right of the window.

  • Install the Code Coverage Package.

Opening Code Coverage Window

  • Open your Unity project.

  • Using the top bar, navigate to Window > Analysis > Code Coverage.

  • The Code Coverage window should appear.

  • You may see a warning requiring you to switch to debug mode to use code coverage. To solve this, switch to debug mode by clicking the bug icon at the bottom right of your Unity project.

  • NOTE: to exit debug mode, click on the bug icon again and select release mode.

  • Make sure the “Enable Code Coverage” box is checked.

  • Ensure HTML Report, Report History, Summary Badges, Additional Metrics, and Auto Generate Report are checked. Auto Open report is optional but recommended.

  • Now you are ready to check your code coverage. Make sure to update your gitIgnore before updating to main.

Updating the GitIgnore

Now that you have added the Code Coverage Package you need to update the gitIgnore to avoid filling the main branch with merge issues. To do so follow the steps below:

  • Locate the gitIgnore file, this should be found in the root of your unity project.

  • Under the lines containing folder references (“/Example”), add this line as a new line:

/[Cc]ode[Cc]overage/

  • Save the file and you are done!
