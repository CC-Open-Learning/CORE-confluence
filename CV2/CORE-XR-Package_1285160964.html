CORE Framework : CORE XR Package

Created by Jacob Martens, last modified on Aug 14, 2024

Core-XR is the CVRI built package for adding VR support to Unity Projects. It is built with OpenXR and XR Interaction Toolkit with a focus on cross-platform compatibility. It is designed to be easy to use, installing just core-xr will get all required dependencies. Adding XR to a scene just requires adding the XR System prefab.

Initial Configuration

After installing core-xr to a project, some configuration steps still need to be followed. Follow the Configuring Project section from the Project Setup article.
