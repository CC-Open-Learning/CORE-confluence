CORE Framework : How to Build and Run a Web-GL Project with SCORM Outside of eConestoga

Created by vtummillo, last modified by Blake Hadaway on Sept 28, 2023

This page will document the steps needed to build a project with SCORM outside of eConestoga successfully.

 Instructions

  1. First Follow the steps for Configuring SCORM properties here Building for WebGL + SCORM Follow those steps until you finish the “Compress Format has been set to Disabled“

  2. Next, Instead of the Enable Exceptions setting being set to “Explicitly Thrown Exceptions Only”, this must be changed to “Full Without Stacktrace” as shown below.

  3. After this step has been completed, you can proceed with the instructions on Building for WebGL + SCORM and create the build. Once that is finished you will be able to locally host the Unity project outside of eConestoga.
