CORE Framework : Azure PlayFab Overview

Created by Lisa Nguyen, last modified by Joshua Moore on Jun 01, 2023

Pros

Cons

-Pricing, Free to start with and use. 10 Titles in “development” mode with a max of 100k unique users. Then pay as you go for “live” titles. As the tool is meant for very high volume tasks(games with 100’s of thousands/millions of unique users) and 10’s of thousands daily users, for VAR Lab I expect the pricing to be very reasonable.

-Game based tool, results in many unnecessary features we have no need for. (Revenue related tracking tools, account linking to steam, maybe segments/AB testing, multiplayer system, voice chat, etc. )

-Powerful out of the box, without a lot of setup. Easy to integrate(for unity).

-Not the most advanced data visualization/processing tools. some auto generated KPI, and allows for the setup of certain queries in a basic and advanced data explorer in graph/chart form. mitigated by the fact if needed we can export data for further processing.

-Good “industry standard” KPI(key performance indicators) tracked out of the box, examples: Average session length, number of unique users, daily amount of users, also auto generates daily and monthly reports of these KPI.

-Good custom abilities, to set up custom events for any needs of the individual title, can even enable cloud scripting based off certain events/rules to run custom code.

-Accessible data, allows export of data collected to CSV, Excel, or Power BI.

-Many integration options available, from engines, to languages.

Features

  • Variety of integrations available, Unity, C#, JS, NodeJS, all of possible value to VARLAB.

  • Automatic generated reports that show key KPI such as daily users, session time, etc.

  • Realtime event stream system, useful for development to see in the manager what is happening in a title in real time.

  • Allows tracking of all sorts of Data throughout a game/title, through out of the box events and the creation of custom events to fit the needs of your title, example being creating a custom progression event for an inspection DLX that tracks how far they get in and the series of correct/incorrect answers.

  • Seems easy to pick up use but powerful enough to allow deep customization

Pricing

Free! up to 10 Titles in development mode, each title getting up to 100k unique users, at which point the account will be required to be upgraded into either a pay as you go or a standard/premium plan. The pricing is per account, account are tied to a studio, which is then tied to multiple titles.
NOTE: Titles have 2 states, in development and live. This allows developers to access titles and play with the tools without running the meter up.
