Helper & Extension Methods

VELCRO UI has various extension and helper scripts which are public static methods. Most helper scripts are used for specific UI prefabs, but VisualElementExtensions.cs, TextElementExtensions.cs, PositionHelper.cs, and StyleHelper.cs have methods for common use cases in many UI scripts.

Show()

Definition

public static void Show(this VisualElement visualElement)

  • Sets DisplayStyle of incoming VisualElement to DisplayStyle.Flex

  • All UI elements inherit from VisualElement

Parameter(s)

none

Example

/// <summary>
/// Shows the root of the information dialog and triggers OnDialogShown
/// </summary>
public void Show()
{
    Root.Show();
    OnDialogShown?.Invoke();
}

SetElementSprite()

Definition

public static void SetElementSprite(this VisualElement element, Sprite sprite, bool hideIfNull = true)

  • Sets the VisualElement.style.backgroundImage property of the incoming VisualElement to the incoming sprite

  • If the incoming sprite is null, the VisualElement will be set to DisplayStyle.None by default, but the hideIfNull bool can be changed if desired.

  • All UI elements inherit from VisualElement

Parameter(s)

Sprite The sprite to set

hideIfNull Whether or not to set the incoming VisualElement to DisplayStyle.None if the incoming sprite is null

Example

/// <summary>
/// Populates the information dialog with content from the incoming scriptable object
/// </summary>
/// <param name="informationDialogSO"></param>
public void SetContent(InformationDialogSO informationDialogSO)
{
    //This will hide the visual element if sprite is null
    imageElement.SetElementSprite(informationDialogSO.Sprite);
    
    //This will not hide the visual element if sprite is null
    imageElement.SetElementSprite(informationDialogSO.Sprite, false);
}

SetBorderColour()

Definition

public static void SetBorderColour(this VisualElement element, StyleColor borderColour)

  • Sets the border colour of the incoming VisualElement to the incoming StyleColor

    • VisualElement.style.borderTopColor

    • VisualElement.style.borderBottomColor

    • VisualElement.style.borderLeftColor

    • VisualElement.style.borderRightColor

  • All UI elements inherit from VisualElement

Parameter(s)

StyleColor The color to set

Example

/// <summary>
/// Updates the progress bar in the task list. Reset inline styling applied 
/// to progress bar before the new value is set
/// </summary>
private void UpdateProgressBar()
{
    progressBarFill.SetBorderColour(StyleKeyword.Null);
    //Set value
}
