CORE Framework : Grabbable

Created by Geyang Liu, last modified by Anzhelika Kostyuk on Mar 06, 2023

In order to let the Hand grab an object, the object must have the Grabbable component. Any rigidbody object can be grabbable, as long as the grabbable is added to or under a rigidbody object.

For more information on this component, see https://earnestrobot.notion.site/Grabbables-9308c564e60848a882eb23e9778ee2b6 .
