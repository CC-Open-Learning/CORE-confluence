CORE Framework : CORE XR

Created by Jacob Martens, last modified on Aug 14, 2024

This section contains documentation related to the core-xr package. This is the currently maintained CVRI XR package. The articles also cover general steps to configure a Unity project for XR.

This guide is based around OpenXR v1.10.0 and XR Interaction Toolkit v2.5.4. Both of these packages are changing in Unity 6, this guide and the core-xr library will need to be updated as Unity 6 becomes more documented/reliable.

This guide assumes some background knowledge on Unity, including creating projects, installing packages, and installing multiple build platforms. This guide also assumes configuration will be done on a fresh Unity project, but the steps will work for previously existing work.
