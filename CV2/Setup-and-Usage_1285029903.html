CORE Framework : Setup and Usage

Created by Jacob Martens on Aug 14, 2024

XR-Only Project

Setup for an XR only project is very simple. After installing core-xr, follow Configuring the Project, then place the XR System prefab in the scene where you want the player to start.

XR and Flatscreen

For a cross-platform project, begin by following the steps for an XR-Only project. Then, you need to add the PlatformController to an object in scene and configure it

Configuring PlatformController

On the same GameObject with the PlatformController, add the IPlatform implementations for every required platform. To every IPlatform implementation, add all platform-dependant objects to the relevant Platform Objects list. Every object in these lists will be enabled/disabled as its platform is enabled/disabled. In the inspector, drag the platform you want the game to start in to the Initial Platform setting of the PlatformController script.

Though any object can be placed in the Initial Platform setting, if it is not an IPlatform then PlatformController will disable itself

Configuring PanelController

In a cross-platform game, consider attaching a PanelController to all FlexXR panels in the scene. They will all be found by the PlatformController, allowing all runtime FlexXR Panels to be updated alongside the active platform.
