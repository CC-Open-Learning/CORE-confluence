CORE Framework : Autohand Player

Created by Geyang Liu, last modified by Anzhelika Kostyuk on Mar 06, 2023

This gameobject handles the player movement and control activities. It requires the AutoHand Player and XR Hand Player Controller Link components.

AutoHand Player

This component handles the properties of the player. Assign the playerâ€™s head camera, tracker offset, and hand gameobjects in the field. Properties such as movement speed, climbing and pushing are customizable here.

XR Hand Player Controller Link

This component links the controller inputs to the player. Assign gameobjects with the XR Hand Controller Link component(the hand gameobjects by default) in the Mve Controller and Turn controller fields. They will be used to navigate the player.

For more information, see https://earnestrobot.notion.site/earnestrobot/Auto-Hand-f78a404f6baf4d85ab705c0d1f92c30e.
