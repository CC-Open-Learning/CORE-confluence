Splash Screen

The splash screen is an image overlay of the entire screen with adjustable messages to hide the initialization of your DLX until the learner is ready to interact with the sim.

Setting up your scene

To get started with implementing the splash screen in your scene

  1. Drag in the splash screen prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Create a SplashScreenSO, and reference it in the prefab. Right Click/Assets > Create > ScriptableObjects > SplashScreenSO

  4. Hook up UnityEvents as required

    1. You will need to call HandleTaskComplete() or HandleAllTaskComplete() to increment the message list and complete the splash screen fade out process

Definition:

public void HandleTaskComplete()

Comments:

  • Increments the internal task counter so that Update() will recognize it and display the next step when messageDuration has been reached

HandleAllTaskComplete()

Definition:

public void HandleAllTaskComplete()

Comments:

  • Increment the internal task counter to the last available index so that Update() will recognize it and display all steps without delay
