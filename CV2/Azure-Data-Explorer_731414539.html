CORE Framework : Azure Data Explorer

Created by Jacky Lam, last modified on Jun 14, 2023

Azure Data Explorer (ADX) is a real-time data analysis tool that can query and visualize data. Although PlayFab also has its Data Explorer (basic and advanced), it lacks features like custom dashboards, saving queries. In short, ADX is the full version of the Data Explorer embedded into PlayFab. Therefore, Azure Data Explorer will be our primary tool to analyze DLX data collected with Azure PlayFab.

Tools & Features

  • Querying Data (with Kusto Query Language)

  • Create Dashboards

  • Exporting Data (CSV)
