Extending the CinemachineBrain

The Main Camera in a Unity scene is controlled by the CinemachineBrain component when using the Cinemachine package.

CinemachineBrainExtension

The general pattern for CinemachineBrainExtensions is to have an extension component attached to the CinemachineBrain. Each individual CinemachineVirtualCamera (or vcam) will have a corresponding settings extension attached, to indicate that the vcam supports this feature and to provide optional overridden settings from the defaults.

The available CinemachineBrainExtensions are described in detail below:

  • CinemachinePan

  • CinemachineZoom

  • CinemachinePathMovement

  • CinemachineFade

CinemachineExtension

CinemachineExtensions are attached to vcams to provide two key functions:

  • indicating that the vcam supports the corresponding CinemachineBrainExtension feature, and

  • optionally overriding the default settings specified for the corresponding feature
