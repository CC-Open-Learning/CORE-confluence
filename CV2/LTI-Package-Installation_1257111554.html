Created by Joshua Moore, last modified on Aug 01, 2024

This package, like many of our current packages is hosted on our UPM package registry. There are 2 paths to install the package, manually adding it to the package.json file, or using Unity’s package manager.

Pre-requisite

A pre requisite to installing the package is having our UPM registry added to your Unity project. Please see the following documentation on how to add the registry: https://varlab-dev.atlassian.net/wiki/x/ZIFjLg

Package Manager
