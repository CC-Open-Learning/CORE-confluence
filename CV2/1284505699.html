CORE Framework : COREV2 VR (Archived)

Created by Jacob Martens on Aug 14, 2024

This section contains all documentation related to COREV2 VR. This library is no longer used or maintained, and has evolved into the CORE XR library. All previous documentation is stored here rather than deleting, in case it becomes relevant again.

The https://bitbucket.org/VARLab/corev2_vr/src/master/ package is based on the OpenXR Package, XR Interaction Toolkit, and the https://assetstore.unity.com/packages/tools/game-toolkits/auto-hand-vr-physics-interaction-165323

asset.

Auto Hand developed a set of its own controller library different than the default XR classes. COREv2 VR will be building on top of them for sake of compatibility.

This section will cover concepts of both the native Auto Hand and COREv2.VR libraries.

  • VR player

  • PC Player

  • VR and PC modes

  • Grabbable

  • Grabbable Events

  • Interactable

  • Vehicle

  • Vive Cosmos Plugin

Installation

Open up the Packages/manifest.json file in your project and add this line to dependencies.

"com.varlab.corev2vr": "ssh://git@bitbucket.org/VARLab/corev2_vr.git#upm"

If your PC is not yet configured to access VARLab repositories via SSH, follow the instructions on Setting Up Bitbucket & Sourcetree

before continuing.
