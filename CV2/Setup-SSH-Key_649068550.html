CORE Framework : Setup SSH Key

Created by Jacky Lam on Apr 19, 2023

SSH Key is required for Unity Editor to access VARLab’s Unity Packages in Bitbucket repo. Bitbucket provides a convenient way to add SSH Key to the runner through the Repository settings.

From Repository Settings, go to SSH keys, click Generate it the key pair hasn’t been generated yet.

Add Bitbucket as Known Host

Under Known Host, add bitbucket.org as the host address, then click Fetch and Add Host.

Add the Public Key to your Bitbucket Account

Copy the public key, then go to Personal Settings → SSH Keys, and add the public key here.
