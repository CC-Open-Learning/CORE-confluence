CORE Framework : TrackerOffset

Created by Geyang Liu, last modified on Jul 27, 2022

Tracker offset provides the base offset location of the player. It is altered by input through the XR Hand Player Controller Link component in Autohand Player .

The Vehicle Boarder script is used for boarding vehicles and not a required component.
