Default Dark and Light Themes

VELCRO ships with theme support for all controls, and UI prefabs included in the package excluding notifications which are the same in both themes. All active, hover, selected, etc. colours are all contained in USS variables in the Dark.tss and Light.tss files. The default dark/light mode primary colours are shown below.

Overriding the default themes for DLX

If your DLX requires a different theme, or even just a tiny shift in colour for any control/UI, then you can achieve this by creating a new theme, and changing the USS variable hex code instead of editing UXMLs.

  1. Create a new theme file

    1. Right Click/Assets > Create > UI Toolkit > TSS Theme File

  2. Reference the theme you are trying to override in the Inherited Themes section

    1. When creating a new theme, it must inherit from UnityDefaultRuntimeTheme or inherit from another theme that inherits from it. In this case, Dark.tss/Light.tss inherit from UnityDefaultRuntimeTheme so it is not necessary

  3. Reference your DLX specific style sheets in the Style Sheets section

  4. Click Apply

  1. Copy the colour variables you want to override in the theme you are trying to override, and paste it into the newly created theme file

    1. Here, I am trying to override the default gold colour of the primary button

  1. Change the hex values to the desired colours

  1. Reference your new theme file in the PanelSettings asset

  1. Changes can be previewed by opening the UXML files affected in the UI Builder and switching the theme in the theme drop down at the top right + preview

Overriding the default font in DLX

VELCRO uses Lato font, and imports the font assets when VELCRO is installed in a DLX. However, just like colours, if you want to change the default font that all USS classes use such as the font-weight classes, then simply repeat steps 5-6 in the above section “Overriding the default themes in DLX” after importing new font files.

  1. Copy the font variables you want to override in the theme you are trying to override, and paste it into the newly created theme file

    1. Here, I am trying to override the default Lato font for all font weights

  1. Download the new font files (ttf). For this example, we are going to replace Lato with Roboto.

  2. Unzip the downloaded font files into the desired location

    1. “/Assets/Resources/Fonts & Materials” is the default suggested location

  3. Create font assets from all .ttf files

    1. Select All Font > Right Click > Create > Text > Font Asset

  4. Change the font asset names to the newly created ones
