Info Popups

The info popup UI element has two sizes, but only the small version has a corresponding script and UI prefab. The large version is intended to be used with object viewer, and is a UXML template only. It’s provides information to a user with an image, text, and with a button to play a an audio track. It can be paired with a helper script to automatically position and enable a popup next to an object on screen selected by a user.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=6092-3049&m=dev

Setting up your scene

To get started with implementing a small info popup into your scene:

  1. Drag in the info popup small prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() with either:

      1. A string containing text for the title label, a Sprite for the body image, and a GameObject which it will be displayed beside (passed to the popup for positioning purposes)

      2. An InfoPopupSO object, with its fields populated with the same desired parameters mentioned above

  4. Ensure any game objects you wish to have the popup respond to hold reference to the popup in their script, or ensure that any HandleDisplayUI calls pass the desired game object. When these objects are clicked on (or whichever action you wish), the popup will automatically appear beside that object.

    1. The info popup sample scene has a good example of an implementation of this.

Root VisualElement of the UI

HandleDisplayUI()

Definition:

public void HandleDisplayUI(string titleText, Sprite bodyImage, GameObject gameObject)

 

Comments:

  • This method is intended to be a single access method to set the text (if any is passed) in the popup, the body image, and the GameObject in the scene that the popup needs to be positioned beside.

  • Invokes OnPopupShown since Show() is called after content is populated

HandleDisplayUI()

Definition:

public void HandleDisplayUI(InfoPopupSO infoPopupSO, GameObject gameObject)

 

Comments:

  • Override of HandleDisplayUI method, accepts an InfoPopupSO object to set the title label and body image fields of the popup along with the GameObject in the scene that the popup needs to be positioned beside

  • Invokes OnPopupShown since Show() is called after content is populated

SetContent()

Definition:

public void SetContent(string titleText, Sprite bodyImage)

 

Comments:

  • Populates the info popup content, but does not call Show()

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Scriptable Object

To set up the info popup, you will need to create a info popup scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > InfoPopupSO

  • Alternatively, you can create a SO through code and fill out properties from there using

    InfoPopupSO popupContent = ScriptableObject.CreateInstance<InfoPopupSO>();
popupContent.Title = "Title text";
popupContent.Image = yourSprite;
