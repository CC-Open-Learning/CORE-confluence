CORE Framework : Unity License Activation

Created by Jacky Lam, last modified on Apr 19, 2023

There are a couple steps you need to do to set up a project repository for build automation.

Manual License Activation

Using a Personal License requires manual activation because activating license in command line requires a serial number and Personal License doesn’t have a serial number.

Get activation file from the unity editor by running the manual pipeline GetActivationFile

  1. Go to the Bitbucket Repository > Pipelines > Run Pipeline

  2. Select a branch and select the pipeline “custom: GetActivationFile”

  3. After the pipeline run is successful, download the license request file (with extension .alf) in the artifacts.

Get license file from Unity

  1. Upload the license request file to https://license.unity3d.com/manual to receive a license file.

  2. You need to log in the Unity account and answer some questions.

  3. Download the license file, the file should end with an extension .ulf.

Store the license file in the repository

We want to store the license file as repository variable, to do that we need to first encode the file with base64.

  1. To base64 encode on Windows, open command line and use the following command:

    • certutil -encode inputfile outputfile

  2. Replace inputfile with the path to the .ulf file, example: C:\Users\UserName\Downloads\Unity_v2021.x.ulf

  3. Replace outputfile with your desire path + license.b64, you will need this file later, example:
    C:\Users\UserName\Downloads\license.b64

  4. Next step is to open the license.b64 file with notepad, copy everything in between “-----BEGIN CERTIFICATE-----” and “-----END CERTIFICATE-----”.

  5. In the repository, create a variable name UNITY_LICENSE_FILE_B64, and paste the content into value.

Possible Automation for Activation Process for Personal License

Automate Activation Process: https://github.com/mob-sakai/unity-activate

This automation tool requires node package manager (npm) to be installed to install the tool.

(Issue) Automated License Activation Sometime Require Security Code

The automated license activation is achieved by automating the manual license activation process in a handless, and sometime it will ask you for a security code if your geographic location changed, which is typical for cloud computer. If this happens in the pipeline, there is no way to provide a security code and the pipeline would stall.

Number of Activations Per License

https://support.unity.com/hc/en-us/articles/360021433091-Does-my-licence-allow-me-to-use-the-Unity-Editor-on-multiple-platforms

Unity Personal users have an infinite amount of activations whereas Plus and Pro users only have two activations per license seat.
