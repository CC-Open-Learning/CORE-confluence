Alternative design

Discovery date

Box Shadows

  • Unity does NOT support box-shadows

  • Box shadows are possible through the KamGam/UIToolkitGlow package Glow Element. VELCRO has box shadow templates defined in Figma under Styles/Elevation https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=1101-902&node-type=canvas

  • This is NO LONGER a hard design constraint. Some box shadows, such as inner shadows are unsupported

Dec 2023

Updated Nov 2024

Dimension values in px (pixels) v/s percentages

  • Unity does NOT support REM, or EM units

Dec 2023

Opacity percentages (eg: 85%)

  • Opacity options are really hard to match

  • Opacity applies to all children elements. This can be overridden, however, buttons specifically do not have child elements. The text is a property of the button

  • Change background colors to shades lighter for hover effects instead of putting a label as a child of the button and trying to force it through

  • Background hovers on buttons should be solid

    Hover effects on separate elements like the radio button/checkboxes/toggles can be semi transparent

Nov 2023

Updated March 2025

Color Gradients

  • Unity supports color gradients for text, though not background colors. It doesn't seem the most robust either

    • https://docs.unity3d.com/Manual/UIE-color-gradient.html

  • This is possible in a limited capacity through the Gradient Element custom control. It is used in Timer templates in VELCRO

  • This is NO LONGER a hard design constraint

Jan 2024

Updated Nov 2024

Blur

  • UI Toolkit does NOT support filters such as blur over the UI

  • Instead of using blur just use a transparent background without the blur.

OR

  • Still researching other solutions outside of UI toolkit

Jan 2024

Font Weight

  • UI Toolkit does NOT directly support changing to any font weight in USS classes

  • Font weight can be set by creating custom USS classes pointing to different font definition assets for the different weights

  • Ex. Lato has font weights of 100 (Thin), 300 (Light), 400 (Regular), 700 (Bold), and 900 (Black). We cannot set it to any other value. However you can bold font assets (Thin/Light/etc) in the UI Builder to set it to in between values (It isn’t quite the regular 700 bold), however that value isn’t revealed to us.

OR

  • Use rich text tags to change font weight in the [Text] field

May 2024

Line Height

  • UI Toolkit does NOT support line heights

Dec 2024

1px Bars, Separators, etc.

  • Elements will be resized depending on screen size. Unity will sometimes cull 1px lines entirely due to rounding

  • Lines, separators should all be a minimum of 2px
