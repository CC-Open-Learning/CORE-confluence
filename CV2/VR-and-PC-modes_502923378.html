CORE Framework : VR and PC modes

Created by Geyang Liu, last modified on Aug 17, 2022

The SceneInitializer prefab is used to determine whether to launch the program in VR or PC mode. To use this component, simply place the prefab inside the scene. On startup, SceneIntializer will check if there are any VR devices connected. If there are none, it will launch the game in PC mode by instantiating the PCPlayer prefab. Otherwise, it will activate the modeMenu gameobject. where users can choose to launch the game in PC mode or VR mode.
