Checklist

Elements to consider:

  • Floating Icon buttons (with drop shadows)
  • Input field with button
  • Tab List
  • UI Shell
  • Notification
  • Pop-over
  • Progress Bar States
  • Tags + States
  • Empty state?
  • Status Indicators
  • Content Switcher
  • Flag (dialouge for confirmation and alerts)
  • Forms
  • Range bar (value of brightness/darkness, volume etc)
  • Expandable Tables
  • Filter tags - PHI
  • Calendar/Date Pickers
