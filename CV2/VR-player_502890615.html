CORE Framework : VR player

Created by Geyang Liu, last modified by Anzhelika Kostyuk on Mar 06, 2023

The VRPlayer prefab is used for playing in VR mode. It uses libraries from the Autohands asset for basic navigation, controls and interactions.

This prefabâ€™s components are dependant on the OpenXR Plugin and XR Plugin Management packages. With these pacakges installed, and set the intiailize XR on startup option to true in Unity project settings, you can simply place the VR player prefab into a scene to play in VR mode.

This page will cover the structure and essential components of the VR player prefab.
