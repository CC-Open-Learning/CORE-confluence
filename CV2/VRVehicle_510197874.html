CORE Framework : VRVehicle

Created by Geyang Liu on Aug 18, 2022

VRVehicle is the base class that all concrete vehicle inherits from. It contains the common fields for vehicles, including power, input, and implements the IInteractable interface to allow users to board them in PC mode. How the input values apply to the vehicles are implemented in the concrete classes.
