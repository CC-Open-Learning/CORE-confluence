CORE Framework : Using Meta Quest 3

Created by Jacob Martens on Aug 14, 2024

The Meta Quest 3 supports multiple ways to run apps. You can either stream them from a PC over Meta Quest Link, or run them directly on the headset as installed APK files. Unity has a feature for Android building to deploy directly to a connected device, which also works for the Meta Quest 3 Headset. This guide will cover both processes to run Unity applications on a Meta Quest 3 headset.

Setup

Headset Setup

If prompted to sign in to the headset, the account used must be configured as a developer account. VARLab has an account, ask IT to help you log in with it when checking out a headset.

In the Meta Quest 3 Headset, locate the settings menu on the toolbar (a green background with a white gear). From there, select System and locate Quest Link from the menu. Enable Quest Link. Locate Developer in the menu, and enable all the settings.

Desktop Setup

Install the Meta Quest Link software, and log in with the same developer account used in the previous step.

In the Meta Quest Link Application, locate Settings and select Beta. From this screen, enable Developer runtime features, then enable every option that appears afterword. Click Turn On for every pop up window for each setting.

Connecting the Headset

Connect the headset to your computer using the USB-C port on the headset. If prompted, update the headset drivers. In the Meta Quest Link application, select Devices and Add Headset. Follow through all the prompts, ensuring the headset is on throughout. If the connection was successful, you should see the Quest 3 and Touch Controllers under Devices. Click on Quest 3 and Touch and scroll down to USB Test under Setup and Support in the window that pops up. Follow the USB Test prompts, ensuring it completes successfully with a transfer rate of at least 2.1 Gbps. This is typically achievable with a USB-C to USB-3 cable. If the cable you are using isn't able to meet these specifications, meet with IT about getting a replacement

Running Apps on the Quest 3

Building an APK

With the headset on, connected to the computer, and recognized in Meta Quest Link, you are able to build directly to the headset. In the Unity Editor, open File->Build Settings. Under Platform, select Android. In the options that appear on the right, locate Run Device. Click Refresh, then locate and select the Oculus Quest 3 from the list of devices. In the bottom of the Build Settings window, select Switch Platform. When Unity has finished switching, select Build And Run, naming and saving the .apk file. For the first build it will take some time, subsequent builds will be completed faster. Ensure you are wearing the headset when the build completes, and the APK will be automatically installed and launched

Running Directly with Quest Link

Faster than building the APK each time is running the application through Unity in Meta Quest Link. Put the headset on while it is connected to a computer running Meta Quest Link. If it launches you directly into Headset Quest Link, skip the following step. Open Settings and navigate to System->Quest Link and select Launch Quest Link. In the subsequent popup, locate the connected PC and click Launch. Note: To exit Quest Link and return to the passthrough camera view, select the left-most option on the bottom bar and click Disable.
