Summary

This section describes the setup and usage of the https://bitbucket.org/VARLab/core-cinema package which provides presets and utility scripts that extend the Cinemachine camera controls package for Unity.

The functionalities provided by CORE Cinema are focused mainly on user-driven camera controls around a 3D digital learning experience.

Installation of the CORE Cinema Package

Open up the Packages/manifest.json file in your project and add this line to dependencies.

"com.varlab.corecinema": "ssh://git@bitbucket.org/VARLab/core-cinema.git#upm"

If your PC is not yet configured to access VARLab repositories via SSH, follow the instructions on Setting Up Bitbucket & Sourcetree before continuing.
