CORE Framework : Controller

Created by Geyang Liu, last modified on Jul 27, 2022

The left and right controller game objects contain the tracked pose driver component. It will let the game object follow the movement of connected VR controllers. You can configure the device types, pose, tracking options here.

You can also create child gameobjects under them for an additional offset.
