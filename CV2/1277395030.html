Glow Element

The glow element is a custom control that allows you to add a glow effect to any visual element in the hierarchy. This is used in VELCRO only for box shadows.

Using the Glow Element

To get started using the glow element in the UI builder

  1. Check if any of the ready shadow templates fit your needs. They can be found at https://varlab-dev.atlassian.net/wiki/spaces/CV2/pages/1301217295/UXML+Templates#Box-Shadows

    1. If they do, drag and drop the template container from the Library panel in “Library > Project > Assets/VELCRO UI/Templates/Box Shadows/”

    2. Otherwise, drag and drop a glow/shadow custom control from the Library panel in “Library > Project > Custom Controls > Kamgam > UI Toolkit Glow/”

  2. Set the visual element to be a child on the glow element

  3. Set the glow element's width and height to match your UI’s width and height

  1. Adjust the properties of the glow/shadow in the Attributes panel in the inspector

    1. Box shadows should all use the hex #787878. This neutral grey looks dark in light mode, and light in dark mode without needing to switch colours on theme switch

USS Classes not Applying to Glow element

Sometimes when you apply a class to a glow element, you will see that some properties do not apply. For example, the “.br-5” class from velcro-style.uss sets the border radius of a visual element to 5px. If you were to apply that class to a glow element, the border radius would not be applied.

To fix this, you must have the border radius (or any property that is affected by this) as an inlined property. Generally this is not good practice but so far, this is the only solution to this problem.
