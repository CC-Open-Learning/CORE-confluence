Bitbucket UPM Links

At the moment we are set up to use private Bitbucket repositories and specific branches on those repos as Unity packages.

This allows us to add a link to the repo’s upm branch in our project’s packages to access it. It also makes sure we can download updates when desired or a specific version of the package at any point.

The downside is that every developer needs their Git configured to use SSH without prompting for a password which can take some time as outlined in Setting Up Bitbucket & Sourcetree

CORE Package Registry

There are also https://docs.unity3d.com/Manual/upm-scoped.html that can be used to give us an even simpler setup however it would require a joint effort between the dev team and IT to get set up properly including authentication as shown here https://docs.unity3d.com/Manual/upm-config-scoped.html.
