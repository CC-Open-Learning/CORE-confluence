Timer

The countdown timer is intended to be a N…3...2…1… film leader that can be used to give the learner a moment before a new game mode starts. This is especially useful if the game mode is timed such as ambulance’s seek and find. The countdown timer only has one customization available, which is a dimmed background to bring focus to the UI.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=3100-6049&t=MRXTZUTJQKK1hhUj-4

Setting up your scene

To get started with implementing timers in your scene

  1. Drag in the count down timer prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() with the start time, and canvas dim state

HandleDisplayUI()

Definition:

public void HandleDisplayUI(int startTime = 3, bool isBackgroundDimmed = true)

Comments:

  • This method is intended to be a single access method to reset the timer, starts the timer with the incoming startTime optionally dims the canvas element, and shows the timer

  • Invokes OnTimerShown

SetContent()

Definition:

public void SetContent(int startTime = 3, bool isBackgroundDimmed = true)

Comments:

  • Sets the timer's startTime with the incoming startTime int, and optionally dims canvas element

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Countdown()

Definition:

public IEnumerator Countdown()

Comments:

  • Coroutine to decrement the CountdownTimerElement custom control's current time. This causes the styling to update

  • Invokes OnTimerEnded when the CurrentTime reaches 0

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

ResetTimer()

Definition:

public void ResetTimer()

Comments:

  • Stops all coroutines, sets timer current time to 0, resets the canvas dim state

  • Invokes OnTimerReset

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the timer

  • Invokes OnTimerShown

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Hide()

Definition:

public void Hide()

Comments:

  • Hides the root of the timer

  • Invokes OnTimerHidden

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()
