Legacy Installation

In the Packages/manifest.json file of a DLX Unity project, add the following line to dependencies:

"com.varlab.scorm": "ssh://git@bitbucket.org/VARLab/scorm.git#0.2.0"

If your PC is not yet configured to access VARLab repositories via SSH, follow the instructions on Setting Up Bitbucket & Sourcetree before continuing.
