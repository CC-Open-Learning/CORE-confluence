Create your first dashboard!

First, we’re going to want to create a blank report. To do this, open the Power BI desktop app and click the blank report button:

Getting data from Azure Data Explorer

After that, you will see your blank report. The next step is to connect to Azure and get our data. For this, click the Get Data button on the top ribbon bar in Power BI:

Connecting to Azure Data Explorer

After clicking Get Data, we’re going to want to type in the search bar “Kusto“, this will bring up the option we want. Click that option then click Connect:

After that, we’re going to want to navigate to Azure, and find our Azure Data Explorer Cluster, and copy the URL for it and paste it into the cluster field.


After that were also going to want to get the database name, and input that into the database field. The database were using right now is “playfab-event-db”. After we input this, we want to select import as the last option. With bigger datasets we would want to create a query to get specific data, but since we have a small amount of data, we can skip that. After were done everything, we can click OK.

Loading data into your Dashboard

It will prompt us to login, which we will do by clicking the sign in button and logging in with our work credentials. After logging in we should see all the data that is in the database we specified (the “playfab-event-db“ database).

Check whatever data we want to use in the dashboard and click load.

Then it will load and now you will have your data in your dashboard!

And it’s as simple as that, you have connected to Azure Data Explorer and pulled data from the database!
