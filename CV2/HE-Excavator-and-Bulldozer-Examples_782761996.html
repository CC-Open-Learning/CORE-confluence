CORE Framework : HE Excavator and Bulldozer Examples

Created by Joshua Moore, last modified by Nick Harrison on Aug 17, 2023

The analytics package has been implemented into the heavy equipment DLX projects both Excavator and Bulldozer. these projects both used Node canvas for the flow of inspections. the metrics tracked by these projects are:

  • if the sim is completed

  • how many inspections have been completed

if you would like to view the details about how this was implemented into both these projects you can view them on the HE implementation documentation. both implementations were the same between projects.
