CORE Framework : Hand

Created by Geyang Liu, last modified by Anzhelika Kostyuk on Mar 06, 2023

The left and right hand gameobjects are responsible for grabbing and interacting with objects. It requires the following components: rigidbody, collider, Hand, and XR Hand Controller Link.

Hand

The Hand component contains the primary features and customizations of the hand itself. It follows the movement of the transfrom in the Follow field, which is linked to the controller.

XR Hand Controller Link

This component is requried to link controller inputs to the hand.

For more information, see https://earnestrobot.notion.site/Hand-967e36c2ab2945b2b0f75cea84624b2f.
