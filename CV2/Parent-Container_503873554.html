CORE Framework : Parent Container

Created by Geyang Liu on Jul 27, 2022

The parent container contains the XR Hand Offset component.

Default device: the relative position to be used for the headset. Do not change this unless you want to rewrite the relative position.

Right and Left Offsets: Assigned the controller tracker transforms here.

Devices: Assign default transfrom and rotation of connected devices.
