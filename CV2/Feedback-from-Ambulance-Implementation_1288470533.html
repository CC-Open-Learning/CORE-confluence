CORE Framework : Feedback from Ambulance Implementation

Created by Richard Colbourne on Aug 15, 2024

Clear

  • How to add packages to the ambulance assembly

  • How to setup the analytics package in the code

  • How to create a PlayFab account

  • Once each part of an event was explained to me (name/key/value) it was easy to understand custom events

    • Custom events are really simple to call

  • Setting the version within PlayFab

    • Easy to understand/do once the process is explained to you

  • Connecting the ADX to PowerBI was easy

Confusing

  • I wasn’t sure where to find the packages folder (have to scroll all the way down the vertical asset view)

    • (Analytics Team Note: Addressed in Analytics V2 Unity Implementation )

  • Had to install the PlayFab package separately and attach it to the ambulance assembly

    • Analytics team also didn’t know that it had to be attached separately

      • (Analytics Team Note: PlayFab has been added as part of the package and no longer requires separate installation - how to attach to project assembly addressed in Analytics V2 Unity Implementation )

  • Not possible to edit the Analytics Configuration SO because it’s part of a package prefab

    • Had to make a new SO and overwrite the reference in the Analytics Service prefab

    • Not the worst thing, but can be a bit confusing

      • 100% needs to be explained

      • (Analytics Team Note: Addressed in https://varlab-dev.atlassian.net/wiki/spaces/CV2/pages/1237155842/Analytics+V2+Unity+Implementation#Creating-the-Analytics-Configuration-File )

  • Which special characters can/can’t be used?

    • Was told about spaces and “_” —> “ “ conversion

    • “:”, “-” and “’” cannot be used

      • (Analytics Team Note: Addressed in https://varlab-dev.atlassian.net/wiki/spaces/CV2/pages/1236467761/Analytics+V2+Code+Implementation#Custom-Events )

  • Syntax for PowerBI measures is confusing

    • Maybe make a confluence page that outlines how to do common ones (eg. total number of an event) or a general overview of common terms?

Other

  • The analytics package was already installed in ambulance (a version from September 2023) so I didn’t have to install it

  • We got a bit lost trying to set up the VersionCheck, but after Richard checked the internal implementation it was an easy fix

    • (Analytics Note: the package has been updated to allow Version Check to be called directly instead of in a callback from Login making it much easier to use).

  • I did not have to connect the DLX PlayFab to Azure Data Explorer, because this was done by the Analytics team
