Toasts

Toasts are simple messages used to provide feedback or additional info to the learner about their next or recent action. Toasts come in simple or complex versions with a hint, location, and custom option for each.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=5817-2030&t=BOXEaocAJnD5aDz4-4

Setting up your scene

To get started with implementing toasts in your scene

  1. Drag in the simple or complex prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() or SetContent() with a ToastSimpleSO/ToastComplexSO object or list of parameters to populate content

FadeIn()

Definition:

public IEnumerator FadeIn()

Comments:

  • Toasts fade in, and bump up based on the properties serialized. On enter, the toast goes from 0 to 1 opacity in fadeDuration seconds and bumps up/down translateDirection units

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

  • Invokes OnToastShown since Show() is called during the FadeIn() coroutine

FadeOut()

Definition:

public IEnumerator FadeOut()

Comments:

  • Toasts fade out, and bump up/down based on the properties serialized. On exit, the toast goes from 1 to 0 opacity in fadeDuration seconds and bumps up/down translateDirection units

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

  • Invokes OnToastHidden since Hide() is called during the FadeOut() coroutine

CloseToast()

Definition:

public void CloseToast()

Comments:

  • Starts the FadeOut() coroutine

  • Called when the X button is pressed

  • Invokes OnToastHidden since Hide() is called during the FadeOut() coroutine

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the toast

  • Invokes OnToastShown

Hide()

public void Hide()

  • Hides the root of the toast

SetContent()

Definition:

public void SetContent(ToastType toastType, string header, string message, string secondaryMessage)

Comments:

  • Calls base class Toast.cs SetContent() to populate the toast type, icon, header, and message, but does not call Show()

Comments:

Scriptable Object

To set up the toast, you will need to create a toast simple or toast complex scriptable object asset.

  • Right Click/Assets > Create > ScriptableObjects > Toasts > ToastSimpleSO/ToastComplexSO

  • Alternatively, you can create a SO through code and fill out properties from there using 

    ToastSimpleSO toastContent = ScriptableObject.CreateInstance<ToastSimpleSO>();
toastContent.ToastType = ToastType.Hint;
toastContent.Header = "Header";
toastContent.Message = "Message";
toastContent.Alignment = Align.FlexEnd;
    ToastComplexSO toastContent = ScriptableObject.CreateInstance<ToastComplexSO>();
toastContent.ToastType = ToastType.Location;
toastContent.Header = "Header";
toastContent.Message = "Message";
toastContent.SecondaryMessage = "SecondaryMessage";
toastContent.Alignment = Align.FlexEnd;

SerializedFields

Field

Description

float fadeDuration

Time in seconds for the toast to go from 0->1 or 1->0 opacity

float translateDirection

Represents the distance and direction which the notification will fade in and back out from its origin. Negative numbers represent up, positive numbers down. Origin is top left

Sprite hintIcon

The background image of the [Icon] element on the left of the toast when the type is hint

The background image of the [Icon] element on the left of the toast when the type is location

Can be set to a sprite of your choice for making a customized toast. Replaces the default icons built into the 2 default toast types
