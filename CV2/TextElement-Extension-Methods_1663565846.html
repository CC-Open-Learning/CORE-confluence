Helper & Extension Methods

VELCRO UI has various extension and helper scripts which are public static methods. Most helper scripts are used for specific UI prefabs, but VisualElementExtensions.cs, TextElementExtensions.cs, PositionHelper.cs, and StyleHelper.cs have methods for common use cases in many UI scripts.

SetElementText()

Definition

public static void SetElementText(this TextElement textElement, string text, bool hideIfNull = true)

  • Sets the TextElement.text property of the incoming TextElement to the incoming text

  • If the incoming string is null or whitespace, the TextElement will be set to DisplayStyle.None by default, but the hideIfNull bool can be changed if desired.

  • All UI elements that have text inherit from TextElement such as Labels and Buttons

Parameter(s)

string The string to set

hideIfNull Whether or not to set the incoming TextElement to DisplayStyle.None if the incoming string is null or whitespace

Example

/// <summary>
/// Populates the information dialog with content from the incoming scriptable object
/// </summary>
/// <param name="informationDialogSO"></param>
public void SetContent(InformationDialogSO informationDialogSO)
{
    //This will hide the label if null or whitespace
    titleLabel.SetElementText(informationDialogSO.Title);
    
    //This will not hide the label if null or whitespace
    titleLabel.SetElementText(informationDialogSO.Title, false);
}

SetElementFontSize()

Definition

public static void SetElementFontSize(this TextElement textElement, FontSize fontSize)

  • Sets the TextElement.style.fontSize property of the incoming TextElement to the incoming FontSize enum

  • All UI elements that have text inherit from TextElement such as Labels and Buttons

Parameter(s)

FontSize The font size to set

//Supported font sizes
public enum FontSize
{
    Medium = 24,
    Large = 28
}

Example

/// <summary>
/// Populates the notification with content from the incoming scriptable object
/// </summary>
/// <param name="notificationSO"></param>
public void SetContent(NotificationSO notificationSO)
{
    messageLabel.SetElementFontSize(notificationSO.FontSize);
}

SetElementColour()

Definition

public static void SetElementColour(this TextElement element, StyleColor colour)

  • Sets the TextElement.style.color property of the incoming TextElement to the incoming StyleColor

  • All UI elements that have text inherit from TextElement such as Labels and Buttons

Parameter(s)

StyleColor The color to set

Example

/// <summary>
/// Populates the notification with custom styles that don't match the default 
/// success, info, or error states
/// </summary>
public void SetCustomNotification()
{
    messageLabel.SetElementColour(customTextColour);
}
