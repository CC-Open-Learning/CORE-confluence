CORE Framework : Network Manager

Created by Lisa Nguyen, last modified on Feb 09, 2023

The Network Manager is used to configure your network and connect the clients and server.

In the project tab search for the ‘Network Manager’ prefab and drag it into the editor, then unpack the prefab.

Network Managers must not contain a Network Object script or be a child of an object that contains it.

By default a Network Manager contains the following scripts:

  • Network Manager

  • Observer Manager

  • Player Spawner

When entering play mode the Network Manager will automatically add:

  • Debug Manager

  • Transport Manager

  • Server Manager

  • Client Manager

  • Time Manager

  • Scene Manager

  • Rollback Manager

  • Network Writer Loop

  • Network Reader Loop

  • Tugboat

All of these can be added manually to allow for custom configurations, more on each manager’s components can be found here: Managers - Fish-Net: Networking Evolved (gitbook.io)
