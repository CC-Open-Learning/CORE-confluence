Helper & Extension Methods

VELCRO UI has various extension and helper scripts which are public static methods. Most helper scripts are used for specific UI prefabs, but VisualElementExtensions.cs, TextElementExtensions.cs, PositionHelper.cs, and StyleHelper.cs have methods for common use cases in many UI scripts.

ToggleTransitionProperties()

Definition

public static void ToggleTransitionProperties(VisualElement element, float newOpacity, float translateDirection)

  • Sets the VisualElement.style.opacity and VisualElement.style.translate properties of the incoming VisualElement to allow for a fade bump transition on enter/exit

  • All UI elements inherit from VisualElement

Parameter(s)

VisualElement The element to apply the style changes to

float newOpacity value of the desired opacity to end at

float translateDirection value of the desired end position to apply. Pixel count and direction. Origin is top left

Example

/// <summary>
/// Notifications fade in, and bump up based on the properties serialized in the script. On enter, the 
/// notification goes from 0 to 1 opacity and bumps up translateDirection units
/// </summary>
public IEnumerator FadeIn()
{
    Show();
    StyleHelper.ToggleTransitionProperties(notification, FadeInOpacity, translateDirection);
}
