CORE Framework : How to Use Moodle to build Web-GL SCORM DLX projects outside of eConestoga.

Created by vtummillo, last modified on Sept 29, 2023

This page will detail the steps needed to build a Web-GL SCORM DLX project for testing prior to uploading to eConesotga.

If you encounter any issues with the steps necessary to create a working - please see this page for troubleshooting.

  1. Follow the steps here to create a working SCORM object for a WebGL build.

  2. Go to Moodle and sign in using the following link and credentials:

Moodle Link: http://moodle.canadacentral.cloudapp.azure.com/

Username: admin

Password: Conestoga_F22

  1. Select My courses > VARLAB Sims

  2. Enable edit mode

  1. Select Add an Activity or Resource

  2. Select SCORM Package

  3. Name your project, and drop the .zip file from step 1. Click Save and Return to Course

  4. You are now able to select the project from the My courses dashboard
