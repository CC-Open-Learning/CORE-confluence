Summary

The CORE Interactions package provides a standardized approach to assigning and responding to mouse events on GameObjects in Unity.

Interaction System

Events

MouseEnter

MouseExit

MouseDown

MouseUp

MouseClick

Interactable

The Interactable component exposes the set of mouse events as serializable UnityEvents. These events can be subscribed to through code in other components, or serialized from the Unity Editor.

Interaction Manager

One or more Interaction Managers can be added to a scene in order to capture and filter all mouse events for one or more Interactable objects.

By default, an InteractionManager will see all Interactable components in a scene, and register itself as a listener to all of their events.

Setting a custom list of Interactables in the InteractionManager will constrain event listening to only those objects.

The InteractionManager exposes the same set of mouse events as an Interactable. When invoked, the events provide a reference to the GameObject that was interacted with.
