The functionality of this script has been moved to the DLX Template namespace and handled by the VELCRO-DLX Template AudioManager interaction. This script will be removed from VELCRO in future versions, but will still exist in all projects.

Helper & Extension Methods

VELCRO UI has various helper scripts which are public static helper methods. SettingsMenuHelper.cs is useful for CloudSave support. When loading volume values of the settings menu and AudioManager values, they may need to be converted.

ConvertLinearVolumeToLog()

Definition

public static float ConvertLinearVolumeToLog(float newValue)

  • Settings menu sliders are 0.0f - 1.0f by default. For more info, please read Settings Menus

  • The settings menu outputs OnSliderChanged<string group, float value> events in a log scale 0.0dB to -80dB for an AudioMixer group

  • Converts a value from a linear slider (X - Y) in the settings menu to a logarithmic value (0f to -80f dB) for use in an audio mixer

  • Useful for CloudSave when loading values

 

Parameter(s)

float newValue to convert from slider value to AudioMixer value

 

Example

/// <summary>
/// Invokes the OnVolumeSliderChanged event when the volume slider's value is changed
/// </summary>
/// <param name="evt">The new float value held by the slider</param>
private void VolumeSliderCallback(ChangeEvent<float> evt)
{
    OnVolumeSliderChanged?.Invoke(MixerVolumeTag, SettingsMenuHelper.ConvertLinearVolumeToLog(evt.newValue));
}

 

ConvertLogVolumeToLinear()

Definition

public static float ConvertLogVolumeToLinear(float newValue)

  • Settings menu sliders are 0.0f - 1.0f by default. For more info, please read Settings Menus

  • The settings menu outputs OnSliderChanged<string group, float value> events in a log scale 0.0dB to -80dB for an AudioMixer group

  • Converts a value from an audio mixer log value (0f to -80f dB) to linear for use with a slider (X - Y) in the settings menu

  • Useful for CloudSave when loading values

 

Parameter(s)

float newValue to convert from AudioMixer value to slider value

 

Example

/// <summary>
/// Invoked from <see cref="CloudSaveAdapter.OnInitialized"/>
/// </summary>
public void ApplySettings()
{
    //These events are invoked to update UI to match applied values loaded in AudioManager.cs
    volumeSlider.value = SaveHelper.ConvertLogVolumeToLinear(soundValues[MixerMasterTag]);
    soundEffectsSlider.value = SaveHelper.ConvertLogVolumeToLinear(soundValues[MixerSoundEffectTag]);
    dialogueSlider.value = SaveHelper.ConvertLogVolumeToLinear(soundValues[MixerDialogueTag]);
}
