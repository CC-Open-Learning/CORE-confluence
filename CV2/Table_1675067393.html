Table

Tables are a complicated piece of UI that allow you to neatly organize information in an almost grid-like format. Developers can add their own categories, entries, and columns to display their information.

https://www.figma.com/design/om9GwMhkfeJpcADbKz4ov8/CVRI-Design-System?node-id=4110-1521&m=dev

Setting up your scene

To get started with implementing Table in your scene

  1. Drag in the Table prefab into the scene hierarchy at “/Packages/Velcro UI/Prefabs”

  2. Replace the PanelSettings reference from VELCRO to your project’s PanelSettings asset

  3. Hook up UnityEvents as required

    1. You will need to supply HandleDisplayUI() with the names of columns to add them to the table

HandleDisplayUI()

Definition:

public void HandleDisplayUI(params string[] columns)

Comments:

  • Sets the columns of the table and displays the table to the screen

  • Takes in infinite parameters that represent the columns of the table

  • Calls the Show() method

AddCategory()

Definition:

public void AddCategory(string categoryName)

Comments:

  • Adds a new category to the table

  • Invokes OnCategoryAdded

RemoveAllCategories()

Definition:

public void RemoveAllCategories()

Comments:

  • Removes all categories and their contents from the table

  • Invokes OnCategoryRemoved for as many categories as there were

AddColumn()

Definition:

public void AddColumn(string columnName)

Comments:

  • Adds a new column to the table

  • Invokes OnColumnAdded

RemoveColumn()

Definition:

public void RemoveColumn(int columnIndex)

Comments:

  • Removes a column at the specified index

  • Logs an error if the index is out of bounds

  • Invokes OnColumnRemoved

RemoveAllColumns()

Definition:

public void RemoveAllColumns()

Comments:

  • Removes all columns and their contents from the table

  • Invokes OnColumnRemoved for as many columns as there were

FindCategoryByName()

Definition:

public TableCategory FindCategoryByName(string categoryName)

Comments:

  • Returns the first category found with the specified name if it exists, otherwise returns null

Show()

Definition:

public void Show()

Comments:

  • Shows the root of the table

  • Invokes OnTableShown

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Hide()

Definition:

public void Hide()

Comments:

  • Hides the root of the table

  • Invokes OnTableHidden

  • Exposed if DLX would like to control individual stages of the UI instead of using HandleDisplayUI()

Unity Events

Event

Description

OnTableShown

Invoked when the table is shown to the screen

  • HandleDisplayUI()

  • Show()

OnTableHidden

Invoked when the table is hidden from the screen

  • Hide()

OnCategoryAdded<TableCategory>

Invoked when a new category is added to the table

  • AddCategory()

Parameter(s):

  • The newly added category

OnCategoryRemoved<TableCategory>

Invoked when a category is removed from the table

  • RemoveAllCategories()

  • TableCategory.RemoveFromTable()

Parameter(s):

  • The removed category

OnCategoryExpanded<TableCategory>

Invoked when a category is expanded

Parameter(s):

  • The category that was expanded

OnCategoryCollapsed<TableCategory>

Invoked when a category is collapsed

Parameter(s):

  • The category that was collapsed

OnColumnAdded<int>

Invoked when a new column is added

  • HandleDisplayUI()

  • AddColumn()

  • The index of the newly added column

OnColumnRemoved

Invoked when a column is removed from the table

  • RemoveColumn()

  • RemoveAllColumns()
