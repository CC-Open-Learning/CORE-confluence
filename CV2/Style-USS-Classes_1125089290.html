The classes documented on this page are all contained in the velcro-style.uss style sheet in the VELCRO UI package. Class functional categories include colour, typography, and border modifiers. Their purpose is to adjust the visual appearance of elements without affecting overall structural layout or position.

Typography

Classes for font size, font weight, and headings are named using the format [property]-[value]

Where properties include:

  • fs for font size

  • fw for font weight

  • h for heading

Where value includes:

  • # value of the font size/font weight for that text property

Using .bold or .italic will keep your current font definition and apply an unknown amount of font weight increase/italic increase to the font. This is true even if the currently active font definition is a font weight of 700+ (Bold) or italicized. It is recommended to match Figma designs to font weight classes rather than use .bold or .italic. Read Font in VELCRO & UI Toolkit for more information on font in UI Toolkit.

Borders

Classes for borders are named using the format [property]-[direction][value]

Where properties include:

  • bw for border width

  • br for border radius

Where direction includes:

  • t for top

  • r for right

  • b for bottom

  • l for left

  • tl for top left

  • tr for top right

  • bl for bottom left

  • br for bottom right

Where value includes:

  • # value of the border width/radius for that border property
